# 🎬 Movie Data Analysis Project

## 📌 Project Description

This project performs exploratory data analysis (EDA) on a dataset of approximately 3000 movies. The objective is to analyze movie performance based on profit, return on investment (ROI), and identify patterns related to actors, directors, and producers.

---

## 🎯 Objectives

* Identify the movie with the highest profit along with its cast and crew
* Analyze which language has the highest average ROI
* Extract all unique genres present in the dataset
* Find top producers based on average ROI
* Identify the most frequent actor and analyze their movies
* Explore relationships between directors and their preferred actors

---

## 🛠️ Tools and Technologies

* **Programming Language:** Python
* **Libraries Used:**

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn

---

## 📂 Dataset Information

The dataset contains information about ~3000 movies, including:

* Budget and Revenue
* Genres
* Cast and Crew
* Original Language
* Release Date

---

## 🔍 Analysis Performed

* Data cleaning and preprocessing
* Calculation of profit and ROI
* Extraction of actors, directors, and producers
* Group-based analysis using Pandas
* Identification of top-performing movies and contributors

---

## 📈 Key Insights

* Certain movies generate extremely high profit compared to budget
* Some languages perform better in terms of ROI
* A few producers consistently produce high ROI movies
* Some actors appear frequently across multiple films
* Directors tend to collaborate repeatedly with specific actors

---

## 🚀 How to Execute

1. Upload the dataset in Google Colab or Jupyter Notebook
2. Run all cells in the notebook
3. Ensure required libraries are installed
